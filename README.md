[![Follow me on Product Hunt](https://img.shields.io/badge/Follow%20me%20on%20Product%20Hunt-@nbox-orange?style=for-the-badge)](https://www.producthunt.com/@nbox)



This repository automatically:
- fetches launches for **today** (based on a configured timezone),
- creates/updates a daily Markdown report in `YYYY/MM/DD-MM-YYYY.md`,
- updates this `README.md` with:
  - a “Today” summary + full table,
  - an archive navigation (expandable by year and month).

## Setup

1) Create a Product Hunt **Developer Token**.
2) Add it as a GitHub secret:
   - Settings → Secrets and variables → Actions → New repository secret
   - Name: `PRODUCTHUNT_TOKEN`

3) Repo settings (recommended):
   - Settings → Actions → General → Workflow permissions
   - Select **Read and write permissions**
   - Enable **Allow GitHub Actions to create and approve pull requests**

### Optional env vars
- `PH_TZ` (default: `America/Los_Angeles`)
- `DATE` (manual override): `YYYY-MM-DD`
