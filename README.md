- fetches launches for **today** (based on a configured timezone),

## Setup

1) Create a Product Hunt **Developer Token**.
2) Add it as a GitHub secret:
   - Settings → Secrets and variables → Actions → New repository secret
   - Name: `PRODUCTHUNT_TOKEN`

3) Repo settings (recommended):
   - Settings → Actions → General → Workflow permissions
   - Select **Read and write permissions**
   - Enable **Allow GitHub Actions to create and approve pull requests**

- `PH_TZ` (default: `America/Los_Angeles`)
- `DATE` (manual override): `YYYY-MM-DD`
